== Troubleshooting

**Service won't restart after containment**:
Check SELinux denials with `ausearch -m AVC -ts recent`. The containment playbook sets strict booleans.

**EDA not triggering**:
Verify the rulebook activation is running: check EDA Controller UI → Rulebook Activations → Status.

**Container build fails**:
Ensure the build context has all required files. Check with `ls /tmp/container-build/`.
